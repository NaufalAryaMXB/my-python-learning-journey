# Beginner Todo List Application

A simple, command-line based To-Do list application written in Python. This project serves as a beginner-friendly introduction to fundamental Python concepts such as loops, conditional statements, lists, and user input validation.

## Features

- **Add a Task:** Quickly add new tasks to your to-do list.
- **Delete a Task:** Remove a task from the list by its corresponding number.
- **Show All Tasks:** View all the current tasks you have added.
- **Exit Program:** Safely exit the application.

## How it Works

The application runs in a continuous `while` loop, presenting a menu of options until the user chooses to exit. It stores the tasks in a simple Python list in memory, meaning the data resets each time the program is restarted. 

## Prerequisites

- Python 3.x

## How to Run

1. Open your terminal or command prompt.
2. Navigate to this project's directory.
3. Run the script:
   ```bash
   python todolist.py
   ```
